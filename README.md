## 📁 Μάθημα 1.3: Blink - Η Πρώτη Ανάσα
Σε αυτό το μάθημα μαθαίνουμε να ελέγχουμε ένα LED χρησιμοποιώντας το Pin 13.

### 🔌 Hardware Setup
1. **LED:** Μακρύ πόδι (+) στο Pin 13, Κοντό πόδι (-) στην Αντίσταση.
2. **Αντίσταση (220Ω):** Από το LED στο GND.
3. **Καλώδια:** Σύνδεση με το Arduino Uno.

### 💻 Κώδικας
Μπορείτε να βρείτε το αρχείο `.ino` στον φάκελο `/src`.
